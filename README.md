## Características

| Pedido | URL                    | Devuelve                                             |
|--------|------------------------|------------------------------------------------------|
| GET    | /                      | index.                                               |
| GET    | /cars                  | Todos los autos.                                     |
| POST   | /car                   | Almacena un auto.                                    |
| GET    | /car/{id}              | Devuelve un auto específico.                         |
| GET    | /car/man/{description} | Devuelve todos los autos de una marca en particular. |
